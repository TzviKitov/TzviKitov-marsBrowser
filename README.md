<h1>Initialization</h1>
<p>
Open console, execute : npm install
</p>
<h1>Execution</h1>
<p>Use the configuration in Webstorm (top right 'play' button) or: open terminal,
and execute : node bin/www
</p>
<p>
Then open your browser at http://localhost:3000
</p>
